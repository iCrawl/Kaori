## Features

- Supports searching up to 11 boorus

## Install

```bash
yarn add kaori
```

## Usage

#### Normal usage:

```js
import { search } from 'kaori';

const images = await search('danbooru', { tags: ['cat'], limit: 1, random: true });
images.map(image => {
	console.log(image.fileURL);
});
```
